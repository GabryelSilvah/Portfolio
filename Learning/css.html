Css

Atualizado: 19/07/2023

Css ( Cascading Style Sheets ), é uma linguagem de estilização para página web. É com o css que as páginas web ganham vida. Através dele podemos mudar a cor do texto, a fonte, a cor de funda da página, podemos aumentar ou diminuir o tamanho da letra, da imagem ou do vídeo. Podemos posicionar cada elementar da página em lugares diferentes, podemos criar algumas animações. As possibilidades são muitas, tudo vai depender da sua criatividade.

Para começarmos a fazer essas alterações na nossa página web é necessário dois procedimentos. Primeiro, você deve informar para o arquivo html onde estão as suas estilizações em css. Para fazer isso é muito simples, basta você ir no seu arquivo html, na parte superior, onde fica a tag <hed> </hed> e digitar a tag <link rel="stylesheet" href=" digite aqui o nome do seu arquivo css"</ dentro de hed. Em href, dentro das aspas, digite o nome que você deu ao seu arquivo css e coloque o .css no final (exemplo.css).



Segundo passo, você sabe o que é classe e id? Se você já souber, pode pular esse tópico. Classes e id são os identificadores que podemos criar para cada tag, podemos dizer que é o cpf da tag. Para criar uma classe, basta você digitar class = "exemplo", dentro do início de uma tag, e entre aspas você digita um nome para essa classe. Para criar id, basta escrever id no lugar class.

Na imagem acima é possível observar que foi adicionado uma propriedade de estilização, a propriedade color. Essa propriedade muda a cor da letra. É possível observar na imagem que essa propriedade foi adicionada entre chaves, {color:green}. A estrutura básica do css é exatamente essa. Coloca o nome da classe e digita as propriedades de estilização dentro das chaves. Simples e objetivo.

Propriedades e suas utilidades:

Color:    ;

Muda a cor do texto.

font-family:    ;

Estabelece uma lista de fontes para serem usadas nos textos das páginas.

font-size:    ;

Define o tamanho da letra. Você pode altera o tamanho utilizando, px, rem e % (font-size: 20px;).

font-weight:    ;

Altera a espessura da fonte. Seus valores são, de 200, 600, 900. (font-weight: 900;).

background-color:    ;

Altera a cor de fundo.

border-radius:     ;

Arredonda as bordas do elemento.

margin:     ;

Coloca margem na parte externa do elemento.

padding:     ;

Coloca margem na parte interna do elemento.

border:     ;

Coloca borda no elemento.

A parte alguns têm dificuldade no início, é a parte de centralizar elementos na página, entretanto não é nada complicado fazer isso. Essa dificuldade é porque não entenderam como utiliza algumas das propriedades que será mostrada logo abaixo.



display: flex;

Por padrão, quando adicionamos elementos no html, como textos, imagens, entre outros, eles são adicionados no sentido vertical, um abaixo do outro. O display na opção:flex organiza nossos elementos no sentido horizontal.

justify-content:     ;

Quando adicionamos o display:flex ele nos possibilita utilizar outras propriedades que antes não estavam disponíveis. O justify-content é uma dessas. Ele serve para alinhamos o elemento em partes diferentes da página. Se escolhermos a opção center estamos dizendo que ele será alinhado no centro horizontal, se escolhermos start ele alinha no início horizontal, se escolher end ele alinha no final horizontal.

align-items:     ;

O align-items é mais uma dessas propriedades que só tem como usar após colocar o display: flex. Se escolhermos a opção center estamos dizendo que ele será alinhado no centro vertical, se escolhermos start ele alinha no início vertical, se escolher end ele alinha no final vertical.

flex-direction:     ;

O flex-direction também é mais uma dessas propriedades que só tem como usar após colocar o display: flex. Se quisermos alinhar um elemento, a melhor maneira é utilizar o display:flex. Mas, Quando usamos display: flex, ele automaticamente coloca os elementos na horizontal, mas às vezes queremos que ele continue na vertical porque só usamos o display: flex para poder utilizar as outras propriedades, como justify-content. É para isso que existe o flex-direction. Ele muda a direção do flex. Uma de suas opções é o column que deixa o conteúdo na vertical novamente sem precisar apagar a propriedade do display:flex.

gap:     ;

O gap é utilizado para dar espaçamento entre os elementos. Ele só pode ser usado após declarar o display:flex.

Acho que deu de perceber que são conceitos bem simples e fáceis de entender. Se você não entendeu com a leitura, calma que quando você escrever o código, verá toda a mágica acontecendo, vai fazer total sentindo.

Muito bem, para utilizar o display: flex, você tem que pensar em qual elemento você quer alinhar na tela. O display:flex não é aplicado diretamente na classe do elemento que você quer alinhar, mas sim no que costumam chamar de pai do elemento. Esse pai do elemento é a tag que está guardando nosso elemento filho.

Como pode ser observado, eu queria alinhar minha imagem, que tem o nome da classe (exemplo). Para fazer isso eu coloquei ela dentro de outra tag, porque o display: flex só funciona se for adicionado na classe "pai", que nessa caso tem o nome de (Container_exemplo).

Resumindo, para alinhar o elemento "filho" eu tenho que aplicar o display:flex no elemento "pai". É uma hierarquia, onde você adiciona o display flex em quem tá em cima, mas o efeito acontece em quem tá em baixo.

As primeiras propriedades citas, antes do display: flex não precisam seguir essa regra. Você pode adicionar elas diretamente no elemento que você deseja estilizar.
