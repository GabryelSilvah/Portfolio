Html

Atualizado: 19/07/2023

Html é uma linguagem de marcação que usa tags para dar significado para cada elemento adicionado no código. Em um código html, podemos adicionar elementos como, texto, links, imagens, vídeos, gifs e arquivos como pdf. Para cada um desses elementos existe uma tag especifica. Por exemplo, para inserir um título na página podemos utilizar a tag <h1>. Cada tag existe uma abertura e um fechamento. Fazendo uma analogia, sempre que vamos escrever um texto começamos com uma margem no início do parágrafo dando espaço, para encerrar o texto colocamos um ponto final. Em html, funciona de maneira parecida. Para começar uma tag de título escrevemos assim <h1>, e para encerrar basta escrever o mesmo, porém com uma barra cortando a tag, < /h1>.

Entretanto, existem tags que não precisam de fechamento. Esse é o caso da tag de imagem. Para adicionar uma imagem, basta digitar < img src=” ” > e no meio das aspas duplas você digita o nome da imagem e o tipo dela. Vale lembrar que você precisa ter guardado essa imagem dentro da mesma pasta onde os arquivos do html estar. Caso contrário você não encontrará essa imagem.

O html precisa de uma estrutura básica para funcionar. Essa estrutura terá que ser criada toda vez que você criar um novo arquivo html do zero. Felizmente alguns editores de código que usamos nos ajudam com atalhos para não precisarmos ficar escrevendo toda essa estrutura no início. Se você estiver utilizando o Visual Studio Code, basta digitar o ponto de exclamação (!) e dar enter que a mágica já acontece. Todos os elementos que for adicionado no Html, deve ser adicionado entre a tag de abertura e fechamento do body.

<input type = “

text

” >

Utilizado para captura informações do usuário. Por exemplo, quando você entra em um site e aparece aquela caixinha para digitar o email, senha, nome, data de nascimento, entre tantas outras coisas?! Aquelas caixinhas são tags do tipo <input type = “ ” >. Dentro das aspas de type você vai escrever qual o tipo de informação você receberá. Pode ser text, email, submit, date, tel e passoword.

<label>

 

</label>

Como foi explicado acima, input é uma tag de entrada de dados. Já a label é uma descrição. Normalmente quando você entra em uma pagina de login, tem o campo para preencher o email ou senha, ao lado do campo tem a palavra email ou senha. Então label é essa descrição que acompanha o campo de preenchimento (input).

</form>

<header>

</main>

<footer>

 

</footer>

Essa é uma tag ultilizada para agrupar outras tags. Nesse caso, as que foram usadas para criar o radapé da página.

<article>

 

</article>

Usamos essa tag para agrupar um conjunto de tags, separando cada grupo em artigos.

<section>

 

</section>

Usamos essa tag para agrupar um conjunto de tags, separando cada grupo em seções.

<div>

 

</div>

Antes de existir as tags header, main, footer, section e article, a div era usada para organizar o código. Após declarar a tag div era dado uma classe ou um id para ela com o nome de header, main, footer, section e article. Era assim que se organizava o código. A tag div ainda é muito utilizada. Usamos ela quando as outras tags de organização não são as melhores opções.

Atributos:

class = "

nome_da_classe

"

É usada para criar um identificador (identidade) para algum elemento dentro do html. Através dessa identidade podemos estilizar um elemento no css. Para chamar uma classe no css basta digitar um ponto (.) e depois digitar o nome da classe que você deu ao elemento no html.

id = "

nome_do_id

"

O id é quase a mesma coisa que a class, com uma duas únicas diferença. A primeira, ele é usado quando o elemento não vai se repetir em nenhum outro lugar do código. Equanto a classe usamos quando teremos varios elementos do mesmo tipo quem receberam a mesma configuração de estilização no css. A segunda diferença é que para chamar um id no css utilizamos o simbolo de jogo da velha (#) seguido do nome que demos para o id no html.
