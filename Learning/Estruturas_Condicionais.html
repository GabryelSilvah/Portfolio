Estruturas Condicionais

Atualizado: 19/07/2023

Estrutura condicionais permite que nosso algoritmo tome decisões entre verdadeiro ou falso de acordo com condições que tenhamos estabelecido. No cotidiano fazemos esse tipo de decisão o tempo todo. Por exemplo, se estive chovendo eu levo o guarda-chuva, senão eu o deixo em casa. Outro exemplo, sempre que você está dirigindo e avista um semáforo você faz uma tomada de decisão, se o sinal estiver vermelho, você para, senão você continua. Você acabou de estabelecer duas condições, se a primeira condição for verdadeira você passa, mas se ela for falsa aí você executa a segunda condição que é a de parar. Só que você não precisa se limitar apenas a duas decisões, se você quiser ou for necessário, você pode toma uma terceira decisão. Como é do conhecimento de todos, um semáforo tem o sinal amarelo, que significa atenção. Então você poderia fazer 3 condições, se estiver verde você continua o trajeto, senão, se estiver amarelo você reduz a velocidade, senão, se estiver vermelho você para.



Trazendo isso para a computação e utilizando outro exemplo, imagine que você está criando um sistema de controle de notas dos alunos. Então você está utilizando estruturas condicionais e cria as seguintes condições, se o aluno tirar nota igual ou maior que 5 está aprovado, senão, se ele tirar nota igual ou maior que 3, mas menor que 5 está de recuperação, senão está reprovado.

Em geral, nas linguagens de programação “se” é representado “if”, e “senão” é representado por “else”. Para montar uma estrutura condicional, primeiramente é preciso declarar a variável que vai armazena nossa informação para posterimente ser testada. Após declarar a variável, escreva if, depois escreva entre parênteses o teste lógico (se tal coisa for igual tal coisa, execute a instrução abaixo), e por último, entre chaves {}, o que vai acontecer caso essa condição seja verdadeira.

Estrutura:
If () {};
else if () {};
else {};

Se você achar que é necessário apenas if e else, é válido.

if (nota >= 5 ){
console.log('aprovado')
}

else {
console.log('reprovado')
}

Se você achar que é necessário apenas um if, também é válido.

if (nota >= 5 ){
console.log('aprovado')
}
