Variáveis

Atualizado: 19/07/2023

São espaços reservados na memória para guardar informações que depois poderão ser usadas para exibir essa informações na tela. Imagine uma sala cheia de gavetas, iguais as que tem dentro do cofre do banco. Todas essas gavetas estão vazias e sem nome, então você decide guardar um colar muito valioso em uma dessas gavetas. Então a primeira coisa que você faz antes de guardar o colar, é dar um nome para a gaveta onde ficará o colar. Com esse nome você pode encontrar onde está seu colar quando quiser pegá-lo de lá. Então você deu o nome “meu precioso” para uma das gavetas e guardou seu colar dentro dessa gaveta. Após alguns meses você decidiu retirar seu colar dessa gaveta. Então indo até a sala das gavetas tudo que você precisa fazer é localizar onde está a gaveta com o nome “meu precioso”.

Aplicando essa analogia no mundo da computação seria assim, você vai reservar um espaço na memória do computador para salvar uma informação. Como faz isso? Primeiro, você precisa definir o tipo de variável que armazenará a informação. Em JavaScript, existem dois tipos de variáveis, const, que significa que a informação que está guarda não mudará. E existe o let, que significa que a informação pode ser mudada. Segundo, você precisa criar um nome para este espaço na memória, ou seja, criar um nome para a variável. Pode ser o nome que você quiser, desde que siga algumas regras específicas para criar um nome de variável.

Por exemplo, você não pode criar nomes para variáveis que tenha símbolos como o “@” no meio ou que comece com número. Beleza, você já criou o nome da variável, o que deve ser feito após criar o nome é guarda a informação que você quiser nela. Exemplo: criei a variável com o nome “altura” e guardei o valor 1,60 nessa variável. O que vai acontecer é que toda vez que eu chamar a variável “altura” aparecera na minha tela o valor que está guardado dentro dela, nesse caso "1,60" aparecerá na tela.



Existem tipos diferentes de dados que podem ser armazenados, e cada linguagem de programação vai ter seus nomes específicos para esses tipos de dados. Mas há pelo menos 3 tipos que, em geral, vão ter o mesmo significado.

Existem os tipos que se refere a:
Números (variável que guarda valores numéricos);
String (variável que guarda caracteres, independente se é número, letra ou os dois ao mesmo tempo);
Lógico (guarda valores do tipo verdadeiro ou falso).

Existe outros tipos, mas esses são os mais comuns e mais usados.

Funções

Função é um trecho de código que pode ser reutilizado várias e várias vezes em outros lugares. No dia a dia fazemos muitas coisas que são repetitivas, que bom seria automatizar algumas delas para que ao apertar um botão tudo acontecesse. Mas, na verdade, já fazemos isso, quando, por exemplo, utilizamos o despertador. É provável que todos os dias você precise acorda no mesmo horário. Imagine se todos os dias você precisasse ir lá no seu aplicativo de despertado para configurar os mesmos horários todos os dias. Vamos combinar que é meio chato ter que fazer isso todos os dias, né?!

Se esse fosse o caso, não seria pouco provável que algum dia, em meio a correria do dia a dia, somados ao cansaço, você se esquecesse de configurar. Então o que acontece de comum a todos é que definimos apenas uma única vez um horário para acordar, depois pegamos esse horário definido e aplicamos para os demais dias da semana. Aplicando no contexto da computação, imagine que você está escrevendo um trecho de código. Ao percebe que você está tendo que repetir o mesmo código várias e várias vezes, porque em algum outro momento você precisou fazer exatamente aquilo que você já tinha feito anteriormente. Você então decide criar esse trecho de código dentro de uma função. Nas funções podemos criar nomes conforme a nossa vontade, claro, desde que siga algumas regras específicas. Dentro dessa função que foi criada podemos escrever nosso código e utilizá-lo mais tarde quando for necessário, e para isso, tudo que teremos que fazer é chama a função pelo nome que foi dado a ela sem precisar reescrever todo o código novamente.

De certo modo, funções são um pouco parecidas com as variáveis, já que é possível definir um nome conforme a sua vontade e é possível armazenar dados nelas. Apesar da semelhança, elas servem para coisas diferentes. Também é importante saber que podemos criar variáveis dentro das funções.

As funções sempre têm dois parênteses ( ) após o nome que criamos para ela, e logo após, abrimos chaves { }, onde escreveremos nosso trecho de código.
